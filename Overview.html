This project implements a data structure known as a 'deque,' for double-ended queue.  It allows constant time access for any element, as well as constant run-times for adding to the front or the back, unlike ArrayList or LinkedList.  The actual implementation involves an array of arrays, rather than a single array, with empty arrays above and below the data to allow room to grow.
